# Summary as Popup Window with Header Button

## What
Change the startup status summary table (from REQ-011) so that it is displayed in a pop-up/modal window rather than as a pane in the dashboard. Add a "Summary" button in the header bar (alongside "Stop Server" and "Configure") so the user can open it at any time, not just on startup.

## Context
The status summary should not take up permanent screen real estate on the dashboard. It should be a closeable popup/modal that the user can dismiss after reviewing and re-open whenever needed via the header button. This replaces the original REQ-011 approach of showing it as a startup landing view.
