# Custom Mac App Icon

## What
Create an icon for the Mac app that resembles the Claude icon but with the rounded square outline rotated 45 degrees to appear as a rounded diamond shape. Install it as the app's desktop/dock icon.

## Context
The icon needs to be generated as a macOS `.icns` file (or equivalent format that electron-builder accepts) and configured in the electron-builder config in `package.json`. macOS icons typically need multiple resolutions (16x16 through 1024x1024). The design should keep the Claude sparkle/asterisk motif inside a rounded diamond instead of the standard rounded square.
